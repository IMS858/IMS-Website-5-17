## Editing content

Each page is standalone HTML. Edit text directly, save, push to GitHub, Vercel auto-deploys.

For copy changes, just open the `.html` file you want to edit. Headlines and body text are plain text inside `<h1>`, `<h2>`, `<p>` tags.
